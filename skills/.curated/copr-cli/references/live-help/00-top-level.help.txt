    whoami              Print username that the client authenticates with

    new-webhook-secret  Regenerate webhook secret for a copr.
    list                List all the copr of the provided
    mock-config         Get the mock profile (similar to koji mock-config)

    delete              Deletes the entire project
    fork                Fork the project and builds in it
    regenerate-repos    Regenerate repository metadata for a project. Useful
                        when automatic generation is disabled
    list-builds         List all builds in the project
    get                 Get information about project

    buildpypi           Build PyPI package to a specified copr
    buildgem            Build gem from rubygems.org to a specified copr
    buildcustom         Build packages from SRPM generated by custom script
    buildscm            Builds package from Git/DistGit/SVN repository.
    build-distgit       Builds a package from a DistGit repository
    status              Get build status of build specified by its ID
    download-build      Fetches built packages
    cancel              Cancel build specified by its ID
    watch-build         Watch status and progress of build(s) specified by
                        their ID
    delete-build        Delete builds specified by their IDs
    edit-chroot         Edit chroot of a project
    get-chroot          Get chroot of a project
    list-chroots        List all currently available chroots.
    add-package-pypi    Creates a new PyPI package
    edit-package-pypi   Edits an existing PyPI package
    add-package-scm     Creates a new SCM package.
    edit-package-scm    Edits an existing SCM package.
    add-package-distgit

    add-package-rubygems

    edit-package-rubygems

    add-package-custom  Creates a new package where sources are built by

    edit-package-custom
                        Edits an existing Custom package
    list-packages       Returns list of packages in the given copr
    list-package-names  Returns list of package names in the given copr

    build-package       Builds the package from its default source
    edit-permissions    Edit roles/permissions on a copr project
    list-permissions    Print the copr project roles/permissions
    request-permissions
